CommandInfo

data class CommandInfo(val title: String, val description: String, val longDescription: String = description, val aliases: List<String>, val usage: String, val minArgs: Int, val maxArgs: Int)

Immutable metadata that describes a command.

This includes a human-friendly title, short description, a detailed longDescription, a list of aliases used to invoke the command, usage syntax in usage, and the expected argument cardinality via minArgs and maxArgs.

Instances of this class are typically provided by command implementations via CommandImpl.info.
