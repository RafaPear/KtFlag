CommandRegister

class CommandRegister<T>(cmd: CommandImpl<T>, helpCmd: CommandImpl<T>? = null)

Registry that holds and resolves command implementations by alias.

By default, the registry auto-registers a built-in help command that lists all available commands and shows details for a specific command. A custom help command can be provided via helpCmd.

Parameters

T

The type of the optional execution context shared by commands.

helpCmd

Optional custom help command; when null, a default implementation is registered.
